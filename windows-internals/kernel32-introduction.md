# What is Kernel32.dll?

Kernel32.dll is one of the core Windows system libraries that provides access to essential operating system functions.

It acts as a bridge between user applications and lower-level Windows functionality.

## Responsibilities of Kernel32.dll
- Process management
- Memory management
- File operations
- Thread creation
- Console input/output
- System information access

## Common Functions
- CreateProcess()
- OpenProcess()
- VirtualAlloc()
- CreateFile()
- ReadFile()

## Why It Matters in Cyber Security

Understanding Kernel32.dll is important for:
- Windows Internals
- Malware Analysis
- Process Injection techniques
- EDR behavior understanding
- Security Research
- Offensive Security

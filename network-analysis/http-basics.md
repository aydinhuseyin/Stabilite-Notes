# HTTP Basics

HTTP (HyperText Transfer Protocol) is the foundation of communication between clients and web servers.

## Common HTTP Methods

- GET → Retrieve data
- POST → Send data
- PUT → Update data
- DELETE → Remove data

## Important Components

### Headers
Used to transfer additional information between client and server.

Example:
```http
User-Agent: Mozilla/5.0
```

### Status Codes

- 200 → Success
- 403 → Forbidden
- 404 → Not Found
- 500 → Server Error

## Why It Matters in Security

Understanding HTTP is important for:
- Web Pentesting
- API Security
- Authentication Testing
- Session Analysis
- Burp Suite usage
